## Project Description

This project demonstrates the use of Python's basic data structures and operations.

The assignment covers:
- Lists
- Tuples
- Sets
- Dictionaries
- Loops
- Conditional Statements

---

## Tasks Completed

### Task 1: Product Collections (Lists & Tuples)

- Created a list of product names.
- Created a tuple containing product details.
- Printed the required products.
- Added new products to the list.

### Task 2: Categories (Sets)

- Created a set of product categories.
- Added a new category.
- Demonstrated that duplicate values are ignored.
- Checked whether a category exists.
- Displayed the total number of unique categories.

### Task 3: Product Pricing (Dictionaries)

- Created a dictionary of product prices.
- Added a new product with its price.
- Updated the price of an existing product.
- Removed a product safely.
- Calculated the average price of all products.

### Task 4: Combined Operations

- Created a product catalog.
- Mapped categories to their respective products.
- Displayed the products belonging to each category.
